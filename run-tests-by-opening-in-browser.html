Instructions

  1. Solve each challenge by writing a function in the challenges.js file.
  2. Each challenge requires you to write a single function to solve.
  3. It's important to name your function exactly as the challenge specifies. Pay attention to the exact naming provided in each challenge.
  4. The test results default to a summary of all challenges. If you want to focus only on the challenges you haven't solved yet, click on the Failures link.
  5. As you work through the challenges, regularly commit your code and push your solutions to GitHub. Your instructors will periodically check your progress. Please do not move or rename the files - just work on solving the challenges within the given structure.
